# CS336 中文笔记

这是基于 Stanford CS336 公开课程笔记整理的中文版学习资料，目标不是逐句直译，而是做成**更适合中文读者复习、面试、补系统直觉**的一套讲义。

这套笔记尽量做到三件事：

- **讲人话**：把大模型训练、推理、数据、对齐里的关键概念串起来。
- **带图复习**：每讲都保留一个原网页中的代表性配图，方便快速回忆。
- **带代码理解**：对适合代码说明的部分补上最小实现或伪代码，避免只会背结论。

## 使用方式

- 如果你是第一次学，建议按 `01` 到 `17` 顺序阅读。
- 如果你是做面试准备，可以优先看：`03 / 05 / 07 / 09 / 10 / 13 / 15 / 16`。
- 如果你是做工程实现，可以优先看：`02 / 05 / 06 / 07 / 08 / 10 / 14 / 16 / 17`。

## 内容组织

- `00-index.md`：全课程地图
- `01-17`：逐讲中文讲义
