# 对象/生命周期/值语义

## 要点

- C++ 里“对象”不仅是类实例，也包括标量、数组、临时对象等
- 重点掌握：**构造/析构时机**、**拷贝/移动**、**引用折叠**（后续与完美转发相关）
- 值语义：像 `int` 一样“拷贝即独立”，是很多良好设计的目标

## 对象的存储期

- 自动存储期（栈上）：作用域结束析构
- 静态存储期：程序结束析构（顺序有坑）
- 动态存储期（堆上）：`new/delete` 或智能指针管理

## 拷贝/移动与 Rule of X

- Rule of 0：优先让成员自己管理资源（如 `std::vector`），类本身不写特殊成员
- Rule of 5：自管资源时，通常要考虑 析构/拷贝构造/拷贝赋值/移动构造/移动赋值

## 易错点

- 返回引用/指针指向局部变量
- 误用 `std::move`（把还能用的对象提前“搬空”）
- 静态对象析构顺序（跨翻译单元）
