# 工具链与编译模型

## 要点

- 源码经 **预处理 → 编译 → 汇编 → 链接** 生成可执行文件
- C++ 的“编译期/链接期/运行期”错误要能快速定位
- `#include` 是文本替换；头文件保护/`#pragma once` 解决重复包含

## 编译流程速记

- 预处理：宏展开、条件编译、展开 `#include`
- 编译：语法/语义分析，生成目标文件（`.o/.obj`）
- 链接：把多个目标文件 + 库拼起来（静态/动态）

## 常见问题

- 多重定义（ODR）
- undefined reference / unresolved external symbol
- 静态库 vs 动态库：链接时机、分发方式、符号可见性
